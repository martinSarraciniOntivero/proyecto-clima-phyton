# proyecto-clima-phyton
Proyecto dedicado a un bot de clima con interaccion 24/7 desarrollado con phyton

Este proyecto nace desde la necesidad de contar con un proyecto propio que ponga a prueba mis conocimientos obtenidos en la facultad. Las buenas practicas de programacion, diseño legible, codigo estructurado.
la idea es que sea interactivo desde la plataforma de telegram, que pueda consultarle pronostico actual, pronostico dia completo y pronostico semanal, en la ubicacion que el usuario/cliente desee.
